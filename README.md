## Project setup
```
npm i
```
### API Key
- Создать Api Key из [OpenWeatherMap](https://openweathermap.org). 
- Введите код в строку **src/store/store.js**
```javascript
apiKey: "Твой ключ"
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
